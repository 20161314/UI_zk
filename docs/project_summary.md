## 1. 项目概述

本项目旨在开发一个名为"助农宣传"的应用（或小程序），利用 AI 技术帮助用户（农户或推广人员）快速生成农产品的宣传内容（文案、图文排版），并支持发布到多个社交媒体平台，同时提供数据分析和电商运营指导，最终助力农产品销售。

## 4. 近期主要变更 (截至 2024-07-31)

*   **删除会员特权页面:** 移除了 `membership.html` 页面及相关功能模块，为项目转化为 Vue 框架做准备。
*   **移动端适配:** 重点优化了 `index.html`, `home.html`, `upload.html` 的移动端显示效果和交互体验，包括 viewport 设置、安全区域处理、响应式布局调整等。
*   **CSS 外部化:** 将 `preview.html`, `publish.html`, `upload.html` 中的内联 `<style>` 标签内容迁移到了对应的外部 CSS 文件 (`preview.css`, `publish.css`, `upload.css`)，提高了代码的可维护性和性能。
*   **发布流程简化:** 根据需求，移除了 `publish.html` 和 `preview.html` 中复杂的自动发布逻辑，现在 `publish.html` 的"发布内容"按钮仅用于触发一个成功提示。
*   **脚本链接:** 将 `js/upload.js` 脚本正确链接到了 `upload.html`。
*   **文档更新:** 更新了本项目总结文档，包含了所有已知的 HTML 文件和更准确的功能状态，计划添加 Vue 项目迁移指南。

## 设计特点

1. **一致的视觉语言**
   - 使用Tailwind CSS构建的统一UI组件
   - 绿色为主色调，体现农产品的自然和健康
   - 卡片式设计和圆角元素，增强友好度

2. **用户体验优化**
   - 进度指示器和状态反馈，提高用户操作确定性
   - 模块化布局，信息层级清晰
   - 动画过渡效果，提升交互体验
   - 底部固定导航栏，确保核心功能易于访问

3. **响应式设计**
   - 适配手机和平板设备
   - 合理的间距和字体大小
   - 触摸友好的按钮和交互区域

- JavaScript 实现交互效果和状态管理
